# Disease Prediction ML

This is a beginner-level machine learning project created for learning purposes.

## Project Structure

Disease-prediction-ML
│
├── README.md
├── requirements.txt
└── src
    └── train_model.py

## Description

This project demonstrates how to:
- Create a basic project structure
- Run a Python file from the terminal
- Load simple data using pandas

Currently, the project only tests data loading and execution.
Machine learning models will be added later.

## How to Run

1. Open terminal in the project folder
2. Install required packages:

pip install pandas

3. Run the program:

python src/train_model.py

## Output

If everything works correctly, you will see:

Code ran successfully
